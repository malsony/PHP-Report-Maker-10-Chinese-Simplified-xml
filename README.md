# PHP Report Maker v10 简体中文语言包

## 使用方法：
1. 将`chinese.xml`文件复制到PHPMaker安装路径的`languages`目录下；
2. 重新启动PHPMaker，在PHP标签的`Default Language`中选择`简体中文`；
3. 去爽吧！
